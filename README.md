### 1. 프로젝트 개요

> 캠퍼스 쉐어는 교내 운동 시설 및 장비 대여의 복잡한 절차를 간소화하고, 예약/인증/알림을 통합 제공하는 iOS 기반 통합 운동 지원 앱입니다.
> 

---

### 2. 기획 배경 및 목표

✔ **배경**

- 운동 시설/장비 이용 시 비효율적인 수기 신청 및 정보 부족
- 사용자의 접근성과 만족도를 높이기 위한 통합 시스템 필요

✔ **목표**

- 직관적인 UI를 통해 간편한 시설/장비 예약 기능 제공
- QR 기반 인증 시스템 구현
- 실시간 맞춤형 알림 기능으로 사용자 편의 향상
- OAuth2 및 JWT 인증으로 보안성 강화

---

### 3. 주요 기능

| 기능 | 설명 |
| --- | --- |
| **시설 예약** | 원하는 시간에 운동 시설 예약 가능 |
| **장비 대여** | 장비 리스트 확인 후 간편하게 대여 |
| **QR 인증** | 예약자 본인 인증을 위한 QR 코드 생성 |
| **알림 기능** | 예약 시간 전 자동 알림 제공 |

### 5. 본인의 역할 및 기여

- **전체 iOS 앱 개발 담당**
    - SwiftUI를 활용하여 앱 구조 설계 및 화면 구성 구현
    - 단방향 데이터 흐름(State, Binding)을 이용한 유지보수 용이한 구조 구현
- **기능별 화면 및 로직 구현**
    - 시설/장비 예약 기능, 예약 내역 조회 화면 구성
    - QR 인증 기능 구현 및 서버 연동 처리
    - 알림 기능 구현 (예약 10분 전 리마인더)
- **UI/UX 설계 및 피드백 적용**
    - Figma 기반 디자인을 앱에 반영

### 7. 향후 개선 아이디어

- **복수 장비 동시 대여 기능 추가**
    
    → 팀 운동 시 장비 여러 개를 한 번에 예약할 수 있도록 기능 확장
    
- **운동 커뮤니티 기능 도입 고려**
    
    → 사용자 간 팀 구성, 파트너 매칭, 일정 공유 기능 추가
    
- **통계 시각화 기능**
    
    → 시설/장비 이용률, 인기 장비 등을 차트 형태로 제공
